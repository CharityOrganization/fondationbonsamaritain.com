#### Parameters

| Name           | Typ      | Beschreibung                                                   |
| -------------- | -------- | -------------------------------------------------------------- |
| `access_token` | `string` | The OAuth access token used to authenticate to the GitHub API. |

#### Beispiel

```json
{
  "access_token": "e72e16c7e42f292c6912e7710c838347ae178b4a"
}
```
