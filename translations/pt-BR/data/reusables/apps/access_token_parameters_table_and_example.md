#### Parâmetros

| Nome           | Tipo     | Descrição                                                       |
| -------------- | -------- | --------------------------------------------------------------- |
| `access_token` | `string` | O token de acesso OAuth usado para autenticar na API do GitHub. |

#### Exemplo

```json
{
  "access_token": "e72e16c7e42f292c6912e7710c838347ae178b4a"
}
```
