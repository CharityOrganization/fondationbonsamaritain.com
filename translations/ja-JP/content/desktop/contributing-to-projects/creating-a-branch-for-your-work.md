### ブランチの作成

{% tip %}

**ヒント：**最初に作成するブランチは、ほとんどの場合`master`であるデフォルトブランチに基づきます。 1つ以上のブランチがある場合、現在チェックアウトされているブランチか、デフォルトブランチに新規ブランチを基づく選択肢が与えられます。

{% endtip %}
