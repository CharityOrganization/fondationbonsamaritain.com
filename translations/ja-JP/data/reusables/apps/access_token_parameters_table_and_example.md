#### パラメータ

| 名前             | 種類       | 説明                                 |
| -------------- | -------- | ---------------------------------- |
| `access_token` | `string` | GitHub APIの認証に用いられるOAuthのアクセストークン。 |

#### サンプル

```json
{
  "access_token": "e72e16c7e42f292c6912e7710c838347ae178b4a"
}
```
