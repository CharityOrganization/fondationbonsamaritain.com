#### Parámetros

| Nombre         | Tipo        | Descripción                                                                       |
| -------------- | ----------- | --------------------------------------------------------------------------------- |
| `access_token` | `secuencia` | El token de acceso de OAuth que se utiliza para autenticarse en la API de GitHub. |

#### Ejemplo

```json
{
  "access_token": "e72e16c7e42f292c6912e7710c838347ae178b4a"
}
```
